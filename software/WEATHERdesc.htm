CONTAM Utilities - CONTAM Weather File Creator

Introduction to the CONTAM Weather File Creator

New in version 1.1:

  • Can handle EPW files that are not a full year long.
  • Can handle EPW files that have sub-hourly data.

Ambient weather conditions that impact building airflows as determined by CONTAM include temperature, barometric pressure, wind speed and wind direction and ambient humidity ratio. For steady-state airflow analysis these user-defined values are provided via ContamW. For transient analysis, these parameters vary over time and are provided via CONTAM weather files (WTH file). You can use the CONTAM Weather File Creator to create WTH files.

WTH files are tab-delimited text files that may be readily viewed or modified with a spreadsheet program.

The tool can convert Energy Plus weather files (EPW files) to WTH files. EPW files can be found on the US Department of Energy's web site for Energy Plus.

You can also use the program to create WTH file templates into which you can paste user-defined or measured weather data.

The WTH file format is provided in the CONTAM documentation.

The CONTAM Weather File Creator is a web application that uses the Silverlight browser plugin. For more information about the requirements for running the CONTAM Weather File Creator see the Application Help.

The CONTAM Weather File Creator is located here:
http://www.bfrl.nist.gov/IAQanalysis/software/WEATHERprogram.htm
