CONTAM Utilities - Type98

Type 98 Description

Type98 enables CONTAM to be coupled with TRNSYS ( www.trnsys.com ) Type56 to perform combined energy, airflow and contaminant transport simulations for multizone buildings. Historically, TRNSYS has implemented Type97 that was based on the airflow calculation capabilities of CONTAM and its predecessor program AIRNET [Walton 1989a]. Type97 only provided access to a subset of the airflow network calculations that CONTAM/AIRNET were capable of performing and did not support contaminant calculations.

Type98 provides the continued benefit of the full features of CONTAM as it evolves without requiring a redevelopment of a new type as previously required with Type97. In order to enable this coupling, new socket communications messages have been added that are documented in CONTAM's user guide in Appendix B. This capability is further enabled using a set of NIST-developed tools that facilitate the process of developing the CONTAM and TRNSYS building models to be coupled.
