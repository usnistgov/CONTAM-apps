CFD0 Editor A CFD Editor for use with CONTAM 3.2 August 18th, 2015

CFD0 Editor

The CFD0 Editor software enables you to define and view a CFD domain for detailed airflow and contaminant transport analysis. You can use it to define a CFD domain for either internal-zone or external-zone coupling with an associated multizone CONTAM model, or to define a stand-alone CFD domain for separate analysis with the zero-equation computational fluid dynamics software CFD0 that is provided along with the CFD0 Editor. CFD0 Editor provides for the definition of the domain, grid and boundary conditions and also allows you to view simulation results within a three-dimensional representation of the domain.

The following procedure enables you to save the "CFD0 Editor Setup.msi" file to your computer. This file contains all of the files needed for CFD0 Editor except the Microsoft .NET Framework 4.0 Client Profile. That can be obtained from the Microsoft Website or through Windows Update.

ContamRV

The Contam Results Viewer, ContamRV, is a tool for visualizing contaminant concentration and airflow results of CONTAM simulations. ContamRV displays buildings in SketchPad form very similar to the manner in which ContamW does. SimRead3 CONTAM simulation file conversion command-line application 3.2 Sept 4th, 2015

SimRead3

The main CONTAM result file, the SIM file, is a binary file. In general, binary files are significantly smaller and faster to work with than text files but must be translated by specialty programs in order for their contents to be human readable.

SimRead3 is a utility provided to enable you to create text files from SIM files. It enables you to select the information you want to obtain from the SIM file and export it to tab-delimitted files that can then be easily imported into a spreadsheet program for further analysis.

Download SimRead3 version 3.2: [ simread32.zip] (approximately 120 KB)

1DZRead A CONTAM 1D Zone Data file converter 1.0 Jan 4th, 2007

1D Zone Data file converter

1DZRead is a command line utility that converts 1D zone contaminant simulation data to tab-delimited files. These results can only be generated using CONTAM's Short Time Step contaminant simulation method.

To use the program, place the file 1DZRead.exe into a directory that contains the PRJ and associated 1D result files (RXR and RZ1 files). Drag-and-drop the PRJ file onto the 1DZRead.exe icon and respond to the input prompts which enable you to select the zones, contaminants and number of cells to output. You can then open the tab-delimited file with a spreadsheet program to view the results.

Download 1D Zone Data file converter: [ 1DZRead.zip] (approximately 60 KB)
