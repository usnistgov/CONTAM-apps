LoopDA

Download LoopDA 3.0

The following procedure enables you to save the installer "LoopDA.msi" to your computer.

  1. Click on "[ LoopDA-setup.msi ]" below.
  2. Select "save this program to disk" and click "OK".
  3. Choose or create a folder into which you want to save LoopDA-setup.msi and click "Save". This will save LoopDA-setup.msi to your hard disk.
  4. Open the folder to which you saved LoopDA-setup.msi.
  5. Run the LoopDA-setup.msi and follow the prompts to install LoopDA.
