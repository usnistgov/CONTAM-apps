Download LoopDA 3.0

The following procedure enables you to save the compressed archive "LoopDAz.zip" to your computer. This file contains all the setup files required to install LoopDA on your computer as well as the LoopDA report (supplemental user manual) and sample files associated with the report. These files will be located in the LoopDA installation directory after you install LoopDA.

  1. Click on "[ LOOPDAZ.ZIP ]" below.

  2. Select "save this program to disk" and click "OK".

  3. Choose or create a folder into which you want to save LOOPDAZ.ZIP and click "Save".
    This will save LOOPDAZ.ZIP to your hard disk.

  4. Open the folder to which you saved LOOPDAZ.ZIP.

  5. Unzip the compressed archive.

  6. Open folder to which you unzipped the LOOPDA files.

  7. Click on the file "SETUP.EXE" and follow the prompts to install LoopDA.
