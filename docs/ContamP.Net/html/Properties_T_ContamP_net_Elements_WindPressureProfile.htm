      WindPressureProfile Properties

      Properties

      Adata You may enter up to 16 angle/pressure coefficient datapoints. Pressure coefficients are in the range -1 to 1. The first value must be at a wind direction of zero degrees. The angle zero indicates a wind blowing directly toward the surface in which the opening lies. An identical value at angle 360º is automatically assumed. 90º indicates a wind blowing parallel to the wall from the right, and 270º is from the left. By making these angles relative to the wall, the same profile might be used on different walls around the building without modification depending on building geometry.
      Desc A description of the profile
      Name The name of the profile
      Type 0 = linear interpolation, 1 = cubic spline, 2 = five point curve fit
