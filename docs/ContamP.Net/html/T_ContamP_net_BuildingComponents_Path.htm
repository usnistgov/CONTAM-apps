      Level The name of the level on which this path is located.
      Lim 0 = none, 1 = pressure, 2 = flow: In the design of smoke control systems there are often limits on the maximum and minimum pressure or flow on a path. For example, a stairwell door can require a minimum pressure drop from the stairwell to the zone to prevent smoke entering the evacuation route, but there is also a maximum pressure drop above which some people may not be able to open the door. You may specify either type of limits (default: no limits) and set the values. When you display the results of a simulation, those paths with pressures or flows outside the limits will be highlighted.
      Mult The multiplier is a constant value allowing you, for example, to use a flow element description of one window and a multiplier of 5 to describe 5 windows in a particular wall. If you have specified a "leakage area" flow element, complete the description here by giving the reference area, length, or number of units.
