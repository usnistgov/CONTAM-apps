      Properties

      ambientZoneName the name of the ambient zone - This is for connecting paths to the ambient zone
      AxialD The axial dispersion coefficient for the zone. Note that the zone dispersion coefficient is zone specific and is not necessarily the molecular diffusion properties of the defined contaminants. This coefficient is empirically based and should account for turbulent mixing as well.
      BldgVol 0 = exclude in building volume, 1 = include in building volume; This refers to the calculation of whole building air change rates that you can now have CONTAM perform during simulation. The associated setting is enabled by default, so if you do not want the zone to be included in the whole building volume you should set this to zero. The air change rate and building pressurization calculations will calculate the total rate of airflow from ambient into all of the zones having this set to 1. Typically, you would include all conditioned space in the building volume; for example, you might not want to include an attic or crawl space.
      Cc0 the Initial concentrations of selected contaminant for dynamic (transient) simulations.
      Cdaxis 0 = Normal Zone: well-mixed zone, 1 = 1D Zone: 1D convection/diffusion zone,
      Celldx This is the cell size into which ContamX will sub-divide the zone during simulation if the zone is to be treated as a 1D convection/diffusion zone. If the axis length is not evenly divisible by the cell size, then the cell size will be adjusted as needed. While there is no strict guidance on the selection of cell size, it should generally be on the order of the flow velocity along the axis times the simulation time step.
      Cfd 0 = Normal Zone: well-mixed zone, 1 = CFD Zone: Only one CFD zone is allowed per CONTAM project.Use the CFD0 Editor (available via the NIST Multizone Modeling website) to define the CFD properties of the zone, e.g., zone mesh and boundaries.
      Color Select a zone fill color (-1 - 144) to help you distinguish zones on the SketchPad. You must activate the Color Zones option from the View menu in order to display zone colors on the SketchPad.
      Ctrl The name of the control node whose output signal controls this zone's temperature.
      H1 the first Z-coordinate min as the height relative to the base of the zone.
      H2 the second Z-coordinate as the height relative to the base of the zone.
      Kinr The name of the kinetic reaction which is in effect fo this zone.
      Level The name of the level of which this zone is a part.
      Name This is the symbolic name of the zone. Enter a name up to 15 characters in length. Zone names must be unique within each level of a building.
      P0 The zone pressure when using constant zone pressure.
      RelHt This is the height of the zone above the base height of the level. This is not generally used.
      Sched The name of a temperature week schedule which will control the temperature of this zone.
      T0 This is the value you want to use as the constant temperature of the zone.
      UAD The units for the axial dispersion coefficient.
      UHt The units the relHt.
      UL The units the X,Y, and Z coordinates of the 1D axis
      UP The units for pressure.
      UT The units for temperature.
      UV The units for volume.
      VarM 0 = constant zone contaminant mass fraction, 1= vary zone contaminant mass fraction; Typically you would set this to variable and allow ContamX to calculate the contaminant concentration within the zone during simulation. However, you may wish to set this to constant as a means of creating a simple contaminant source within a zone. If you set this to constant, the zone concentrations will begin and remain at the values you set for the Initial Concentrations (CC0) array.
      VarP 0 = constant zone pressure, 1 = vary zone pressure; The zone air pressure relative to ambient can be specified as either variable or constant. Typically you would set the zone pressure to be variable and allow it to be calculated during the simulation. This option is provided mainly for developers to perform analytic test cases of CONTAM.
      VfNodeName The node name to use from the CVF or DVF file.
      VfType 0 = do not use a values file. 1 = use the project's CVF file, 2 = use the project's DVF file
      Vol Zone volume is used in the dynamic contaminant calculations.
      X1 The X and Y values are in absolute coordinates. These values can be based on scale drawings and/or a user-defined coordinate system. Unlike walls on the SketchPad, axes are not required to be rectilinear, i.e., you may specify axes that are not horizontal or vertical in the x-y plane.
      X2 The X and Y values are in absolute coordinates. These values can be based on scale drawings and/or a user-defined coordinate system. Unlike walls on the SketchPad, axes are not required to be rectilinear, i.e., you may specify axes that are not horizontal or vertical in the x-y plane.
      Y1 The X and Y values are in absolute coordinates. These values can be based on scale drawings and/or a user-defined coordinate system. Unlike walls on the SketchPad, axes are not required to be rectilinear, i.e., you may specify axes that are not horizontal or vertical in the x-y plane.
