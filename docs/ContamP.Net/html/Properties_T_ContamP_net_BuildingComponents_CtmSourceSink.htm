      CtmSourceSink Properties

      Properties

      Cc0 Initial Concentration - for Boundary Layer Diffusion Controlled and Deposition with Resuspension
      Color Set the color (-1 - 144) of the icon. This allows you to distinguish different icons on the SketchPad
      Ctrl The name of the control node whose output signal controls this icon's strength.
      Element The name of the source/sink element associated with this source/sink.
      Hmax Relative Elevation Maximum (1d zone)
      Hmin Relative Elevation Minimum (1d zone)
      Mult A constant value by which the source strength will be multiplied during simulation. With this feature you could define a source/sink element having a source strength per unit area then use the multiplier as the area of the zone for each source/sink that uses the per unit area source/sink element.
      Name The name of the source/sink. In a CFD zone, this name will be used to identify the source/sink within the CFD0 Editor that is used to define the location of the source with respect to the CFD zone.
      Sched The name of the week schedule that will control the icon.
      UXyz Length units for X,Y, Z
      VfNodeName The node name to use from the CVF or DVF file.
      VType 0 = do not use a values file. 1 = use the project's CVF file, 2 = use the project's DVF file
      Xmax Source/Sink X maximum (1d zone)
      Xmin Source/Sink X minimum (1d zone)
      Ymax Source/Sink Y maximum (1d zone)
      Ymin Source/Sink Y minimum (1d zone)
      Zone The name of the zone where the source/sink is located within. Cannot be ambient.
