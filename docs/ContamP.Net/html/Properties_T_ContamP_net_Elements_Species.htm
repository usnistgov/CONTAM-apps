      Species Properties

      Properties

      Ccdef CONTAM will apply this value as the default initial concentration for each zone that you create. You can revise the initial concentration for individual zones as per your requirements. This is also the ambient contaminant concentration during simulations which do not utilize contaminant data from one of the external contaminant data files (CTM or WPC).
      Cp The specific heat of a contaminant will used during simulation only when the Variable Junction Temperatures method is selected.
      Decay This is an exponential decay constant based upon the half-life of the radioactive species
      Desc Use this to provide a more detailed description of the species.
      Dm This value will be used during simulations using the 1D Duct Model of the Short Time Step Method. The dispersal coefficient will be calculated from this value depending on the Reynolds Number in a given duct segment.
      Edens Enter a density that you want CONTAM to use as the effective density of a species you want to consider to be a particulate type species. Currently, CONTAM only utilizes this value when converting between particle count units and particle mass and volume units.
      Kuv A species-dependent rate constant used to determine UVGI filter effectiveness for a specific contaminant
      Mdiam Enter the mean particle diameter if this species is to be a particulate type species. CONTAM utilizes this value when converting between particle count units and particle mass and volume units and to determine filter efficiency of Simple Particle Filter elements for this size particle
      Molwt The molar mass in g/mol or kg/kmol. CONTAM uses this value to convert volumetric concentrations to mass fractions and to calculate the gas constant of the air mixture made up by non-trace contaminants. You must enter a value of at least 1.0.
      Name You will use this name to refer to the species throughout the program. You must give each species a name that is unique to the current project.
      Ntflag 0 = trace contaminant, 1 = non-trace contaminant
      Sflag 0 = not simulated, 1 = simulated
      Ucc concentration units
      Ucp specific heat units
      Udm diffusion units
      Ued effective density units
      Umd mean diameter units
      Used 0 = species is not used, !0 = species is used
