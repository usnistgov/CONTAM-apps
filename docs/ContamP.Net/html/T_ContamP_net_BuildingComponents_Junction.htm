      Properties

      Ad The area of the duct to which the terminal is connected. Typically, this would be the same area as the duct segment just downstream/upstream from the supply/return air terminal. This value is used during simulation to determine the airflow rate through the terminal.
      Af This is the total minimum area of the openings in the air outlet or inlet through which air can pass. This area will be used to calculate the velocity from the airflow rate at the terminal when displaying simulation results on the SketchPad.
      Bal 0 = do not balance this terminal, 1 = balance this terminal
      Cb This is the loss coefficient.
      Cbmax Provide a maximum value for the Balance Loss Coefficient. This will be used to provide warning information if the Balance Loss Coefficient determined by the automated balancing procedure of ContamX exceeds this value.
      Cc0 the Initial concentrations of selected contaminant for dynamic (transient) simulations.
      Color Select a color (-1 - 144) to help you distinguish junctions on the SketchPad. You must activate the Color Ducts option from the View menu in order to display zone colors on the SketchPad.
      Ct Characteristic loss coefficient of the terminal.
      Ctrl The name of the control node whose output is used to control this path
      Ddir direction of terminal duct - to show wind pressure
      Ddname The name of the downward duct associated with the junction/terminal.
      Fdes design flow rate
      Fdir positive flow direction - to show flow
      Filt The name of the filter used for this junction/terminal.
      Icon terminal/junction icon to show on the sktechpad
      Kinr The name of the kinetic reaction which is in effect fo this zone.
      Level The name of the level of which this zone is a part.
      Name The name of the junction. The name must be unique in the project from other junctions.
      P0 junction initial pressure
      RelHt The height of the midpoint of the junction or terminal point relative to the current building level.
      Sched The name of the week schedule that will control this path.
      T0 Enter the temperature you wish the junction or terminal to use during simulation.
      UA Area units
      UDP pressure units
      UF flow units
      UHt relative height units
      UT Temperature units
      VfNodeName The node name to use from the CVF or DVF file.
      VfType 0 = do not use a values file. 1 = use the project's CVF file, 2 = use the project's DVF file
      Vol The volume of the junction/terminal.
      Wind 0 = no wind pressure, 1 = use wind pressure for this path
      WPmod This accounts for differences between wind velocity profiles at the building site and that at a measurement location, e.g. an airport weather station
      Wpp The name of the wind pressure profile that applies to this path.
      WPset A constant wind pressure (wind speed and direction independent).
      X This is the absolute X coordinate for the selected junction/terminal.
      Y This is the absolute Y coordinate for the selected junction/terminal.
      Zn The name of the zone where the junction/terminal is located. ambt for ambient
