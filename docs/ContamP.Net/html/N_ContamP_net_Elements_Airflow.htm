      DdBdf This is the mass flow version of the Backdraft Damper duct flow element. The Backdraft Damper models make it possible to model a feature (e.g. a smoke control damper) that has different flow resistances depending on the direction of the pressure drop, with greatly reduced (or zero) flow in one direction.
      DdBdq This is the volumetric flow version of the Backdraft Damper duct flow element. The Backdraft Damper models make it possible to model a feature (e.g. a smoke control damper) that has different flow resistances depending on the direction of the pressure drop, with greatly reduced (or zero) flow in one direction.
      DdCmf This airflow element allows you to describe the airflow through a duct as a constant mass airflow rate.
      DdCvf This duct flow element allows you to describe the airflow through a duct as a constant volumetric flow rate. The design flow rate you specify is in terms of standard air. CONTAM uses a value of 1.204 kg/m3 based on conditions specified by ASHRAE for dry air: 101.325 kPa and 20ºC [Chapter 18 of ASHRAE 2004]. Therefore, if actual conditions during simulation do not match these standard conditions, the results will differ from specified flow due to differences in air density.
      DdDwc This model uses the Darcy-Weisbach relation and Colebrook’s natural roughness function

      DrPl2 This model accounts for two-way flow due to the stack effect acting over the height of a tall opening. It uses two power law flow models at different heights to approximate a single tall opening.

      FnCvf This model describes an airflow element having a constant volume flow rate. The design flow rate you specify is in terms of standard air. CONTAM uses a density of 1.204 kg/m3 based on conditions specified by ASHRAE for dry air: 101.325 kPa and 20 ºC (ASHRAE 2004, p 18.4). Therefore, if actual conditions during simulation do not match these standard conditions, the results will differ from specified flow due to differences in air density.

      PlTest2 This airflow element allows you to provide two test data points that ContamW reduces to the powerlaw model by calculating the flow coefficient based on input values
      QfCrack This model employs a quadratic relationship of the form DP = aQ + bQ² where Q is the volume flow rate [Baker, Sharples, and Ward 1987]. CONTAM uses the mass flow version of that formula: DP = aF + bF² where DP is the pressure drop and F is the mass flow rate
      QfFab This model allows you to directly enter the coefficients, a and b, into the mass flow version of the quadratic flow model. Where DP is the pressure drop and F is the mass flow rate.
      QfQab This model allows you to directly enter the coefficients, a and b, into the volume flow version of the quadratic flow model. Where DP is the pressure drop and Q is the volumetric flow rate.
      QfTest2 Measured data (2 points) is reduced to the mass flow version of the quadratic model.
      SrJwa CONTAM implements the self-regulating vent as described in [Axley 2001]. This element limits the airflow rate in both directions through a flow path with user-defined limiting air pressure differences.
