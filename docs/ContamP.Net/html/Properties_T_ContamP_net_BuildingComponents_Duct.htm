      Duct Properties

      Properties

      Ain flow area at inlet end - future
      Aout flow area at outlet end - future
      Color Select a duct color (-1 - 144) to help you distinguish ducts on the SketchPad. You must activate the Color Ducts option from the View menu in order to display zone colors on the SketchPad.
      Ctrl The name of the control node whose output signal controls this duct's flow.
      Dir positive flow direction on sketchpad
      Element The name of the duct flow element associated with this duct.
      Filter The name of the filter associated with this duct.
      Jctm The name of the duct junction to which the segment connnects, which is the destination junction for the positive flow direction
      Jctn The name of the duct junction to which the segment connnects, which is the origin junction for the positive flow direction
      Length the length of the duct segment. This will be used along with the cross-sectional area to determine the volume of the duct segment.
      Name The name of the duct. It must be unique among the ducts in the project.
      Sched The name of the week schedule that will control the duct.
      Sllc This is the sum of all dynamic loss coefficients due to junction losses and all fittings in the segment. This term only applies to the Darcy-Colebrook duct flow element model
      UA Area units
      UL Length Units
      VfNodeName The node name to use from the CVF or DVF file.
      VfType 0 = do not use a values file. 1 = use the project's CVF file, 2 = use the project's DVF file
