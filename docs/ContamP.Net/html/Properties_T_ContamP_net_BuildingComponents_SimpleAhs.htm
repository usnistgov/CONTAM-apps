      SimpleAhs Properties

      Properties

      Color Set the color (-1 - 144) of the icon. This allows you to distinguish different icons on the SketchPad
      Ctrl The name of the control node whose output is used to control the OA fraction of this AHS
      Desc Use this to provide a more detailed description of the air handling system.
      ExhPath The name of the exhuast path - output only
      Level The name of the level of which this AHS is a part.
      MinOa This is the minimum outdoor air introduced into the AHS supply flow.
      Name The name used to identify the Air-handling System. Air-handling system names must be unique.
      OaFilt The name of the outdoor air filter which affects the air brought in by the simple air-handling system from outside the building
      OaPath The name of the outdoor air path - output only
      OaSched This is the name of the schedule of the fraction of outdoor air, fo, to be introduced to the simple air handling system
      OaX This is the absolute X coordinate that define the location of the outdoor air intake of the selected air handling system.
      OaY This is the absolute Y coordinate that define the location of the outdoor air intake of the selected air handling system.
      OaZ This is the absolute Z coordinate that define the location of the outdoor air intake of the selected air handling system.
      RCc0 initial contaminant concentrations for the return air sub-system of this simple air handling system
      RecFilt The name of the recirculation air filter which affects the return air being circulated back through the air handler
      RecPath The name of the recirculation path - output only
      RKinr name of the kinetic reaction for the return zone
      Rvol volume of the return sub-system (implicit return zone) of the simple air handler (e.g., return-side duct work etc.). This volume is similar to a zone volume and will be used in the simulation of contaminant transport.
      Rzname the name for the return zone which can be used for defining return paths - output only
      SCc0 initial contaminant concentrations for the supply air sub-system of this simple air handling system
      SKinr name of the kinetic reaction for the supply zone
      Svol volume of the supply sub-system (implicit supply zone) of the simple air handler (e.g., supply-side duct work etc.). This volume is similar to a zone volume and will be used in the simulation of contaminant transport.
      Szname the name for the supply zone which can be used for defining supply paths - output only
      UF flow units
      URV volume units for return zone
      USV volume units for supply zone
      UXyz length units for X,Y and Z
      VfNodeName The node name to use from the CVF or DVF file.
      VfType 0 = do not use a values file. 1 = use the project's CVF file, 2 = use the project's DVF file
