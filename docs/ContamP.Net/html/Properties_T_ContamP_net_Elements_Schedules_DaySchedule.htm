      DaySchedule Properties

      Properties

      Desc Field for entering a more detailed description of the specific schedule.
      Name This is the name you give to this day schedule. This name must be unique among the day schedules within a project.
      Points These are the time/multiplier pairs that you set to define a day schedule. The times are entered as seconds 0-86400. The first data point must be for time = 0. The last datapoint must be for time = 86400. For unitless schedules, the value is a multiplier between 0.0 and 1.0 used to adjust down the maximum design values of various schedulable building components, e.g., constant volume flow rate of a fan flow element.
      Shape 0 = Rectangular: characterized by a step function between the data points, 1 = Trapezoidal: characterized by linear interpolation between the data points.
      Ucnv temperature units for display
      Utyp 0 = no units, 5 = temperature units
