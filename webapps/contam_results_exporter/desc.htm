CONTAM Results Export Tool

Description

The CONTAM Results Exporter Tool enables the user to export all or selected CONTAM results for the entire simulation or a subset of the simulation period.

Features

The CONTAM Results Export Tool has five export options.

  1. Export .sim file to text files: This option exports CONTAM results to tab-delimited text files. The results that can be exported include but are not limited to: airflows, pressure differences, and contaminant concentrations.
  2. Compute temporal/spatial averages of concentrations: This option exports concentration results that are averaged over a specific period of time or averaged over selected zones (or the entire building).
  3. Process exposure results: This option exports exposure results by contaminant or by each occupant exposure object simulated.
  4. Export infiltration to EnergyPlus: whole building air change rates: This option exports whole building infiltration rates to EnergyPlus. More details can be found in Polidoro et al. (2015).
  5. Export infiltration to EnergyPlus: zone-specific air change rates: This option exports zone-specific infiltration rates to EnergyPlus. More details can be found in Polidoro et al. (2015).
